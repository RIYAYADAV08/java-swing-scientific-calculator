# Java Swing Scientific Calculator

A desktop-based Scientific Calculator developed using Java Swing and AWT. The application provides an interactive graphical user interface (GUI) for performing both basic arithmetic and scientific mathematical operations.

## Features

* Addition, Subtraction, Multiplication, and Division
* Trigonometric Functions (Sin, Cos, Tan)
* Square Root Calculation
* Logarithmic Operations
* Interactive Graphical User Interface (GUI)
* Event-Driven Programming using ActionListeners
* Clear and Reset Functionality
* User-Friendly Layout and Design

## Technologies Used

* Java
* Java Swing
* AWT
* Event Handling
* Object-Oriented Programming (OOP)

## Project Overview

This project was developed to strengthen Java GUI development skills and implement scientific calculations through an interactive desktop application. The calculator performs both basic and advanced mathematical operations while demonstrating event handling, user interface design, and object-oriented programming concepts.

## Learning Outcomes

* GUI Development using Java Swing
* Event Handling with ActionListeners
* Object-Oriented Programming Concepts
* Mathematical Computation using Java
* Desktop Application Development
