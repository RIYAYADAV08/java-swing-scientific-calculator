# Java Swing Scientific Calculator

A desktop-based Scientific Calculator developed using Java Swing and AWT. The application provides an interactive graphical user interface (GUI) that performs both basic arithmetic and scientific mathematical operations.

## Features

* Addition, Subtraction, Multiplication, and Division
* Trigonometric Functions (Sin, Cos, Tan)
* Square Root Calculation
* Logarithmic Operations
* User-Friendly Graphical Interface
* Event-Driven Functionality using ActionListeners
* Clear and Reset Functionality

## Technologies Used

* Java
* Java Swing
* AWT
* Event Handling
* Object-Oriented Programming (OOP)

## Project Overview

This project was developed to strengthen Java programming concepts, GUI development, and event handling. It demonstrates the implementation of a scientific calculator with a responsive user interface and mathematical computation capabilities.

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/RIYAYADAV08/Java-swing-scientific-calculator.git
   ```

2. Open the project in VS Code, IntelliJ IDEA, or Eclipse.

3. Compile and run the Java file:

   ```bash
   javac JavaSwingScientificCalculator.java
   java JavaSwingScientificCalculator
   ```
